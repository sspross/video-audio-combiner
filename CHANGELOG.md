## [0.1.0] - Initial Release

### Added
- Desktop application for merging audio tracks from different video sources
- Automatic audio synchronization via waveform analysis
- Onset strength envelope detection using librosa
- Cross-correlation based alignment offset detection
- Waveform visualization in the UI
- Preview generation before final export
- Support for multiple audio track selection
- macOS DMG distribution for ARM64 and Intel
